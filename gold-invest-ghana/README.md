# GoldInvest Ghana - Investment Platform

## Setup
1. Download all files preserving folder structure.
2. Open `index.html` in a modern browser.
3. Register a user account, then login.
4. Access admin at `/admin/login.html` (admin/AdminGold2025).

## Features
- User registration with referral code.
- Dashboard with recharge/invest (Gold image grid, amounts).
- Payment page with network selection, screenshot upload, number 0202666033.
- Admin approves deposits → user balance updates.
- Referral tracking and downline list.

## Future Migration
